# Akara-and-pap
ra and Pap* is a classic Nigerian breakfast combo 🇳🇬🔥   *Akara*  *What it is*: Deep-fried bean cakes made from peeled black-eyed peas or brown beans. The beans are blended into a smooth batter, whipped till fluffy, seasoned with onions, salt, pepper, and sometimes crayfish. Then spooned into hot oil till golden and crispy outside, soft inside.
